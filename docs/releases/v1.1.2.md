# 统票票 v1.1.2

“统票票·智能选票统计系统”首个公开版本。

## 本版亮点

- 自动识别 PDF 选票中的题头、姓名、评价档次和表格布局。
- 本地识别手写“√”，逐页实时累计，不依赖网络。
- 建议复核窗口支持原票定位、高亮、人工改档和空白项批量确认。
- 生成与原选票单栏或双栏结构相匹配的 Excel 汇总表。
- 支持最大 100MB 的多页 PDF。

## 安装

下载并运行：

`统票票·智能选票统计系统-安装程序.exe`

系统要求：Windows 10 / 11 x64。

安装包未使用商业代码签名证书，Windows 可能显示“未知发布者”。请确认文件来自本仓库 Release，并核对以下 SHA-256：

```text
8BBDBF86E35A29A535F5CF90F872D235130A3BD39F5E37CF880F41EEA3775E70
```

## 隐私

OCR、统计、复核和 Excel 导出均在本机完成。软件不上传选票、不调用云端 OCR，也不包含遥测功能。
