# 统票票界面焕新设计

## 目标

以已确认的四张演示图为视觉方向，提升桌面端界面的信息层级和观感，同时保持本地 OCR、逐页统计、人工复核和 Excel 导出的行为不变。

## 主页面

未选择文件时，保留现有品牌头部、大标题和“选择 PDF”入口。选择文件后，主页面替换为进度工作台：

- 顶部显示文件名、页码进度和总体百分比。
- 中部显示已识别人员、评价档次和累计勾选三项实时指标。
- 下部将当前选票预览和本页识别结果并排展示。
- 完成后，在同一页面切换为结果区：人员等次矩阵、档次汇总和 Excel 导出按钮。

所有内容沿用现有的浅暖灰背景、白色品牌区、绿色“票”图标、宽留白和虚线外框；不增加侧边栏，也不改变选票上传、统计或导出接口。

## 导出与复核

- Excel 导出完成后显示轻量成功提示，告知保存位置，不改变现有手动导出机制。
- 建议复核窗口保持原有选票定位、下拉选择、滚轮选择、确认同步和批量处理逻辑。
- 复核窗口仅调整为浅色大留白、虚线外框、明确的人员选中态和绿色确认按钮。
